# 고유가 피해지원금 계산기 (Oil-Relief-Fund)

> 2026 고유가 피해지원금 1·2차 지급 정보를 시민이 빠르게 확인할 수 있는 단일 페이지 도구.
> 내 지역의 지급 등급, 건강보험료 기준 대상 여부, 신청 요일까지 5초 안에 확인.

🌐 **배포 사이트**: [oil-relief-fund.vercel.app](https://oil-relief-fund.vercel.app)

---

## 📌 주요 기능

- **🗺️ 전국 시군구 지도** — 카카오맵 위에 전국 약 250개 시군구 폴리곤을 4단계 등급(특별·우대·일반·수도권)으로 색상 구분 표시
- **🔍 지역 검색·자동완성** — 시·군·구 이름이나 주요 지명 입력 시 등급과 1인당 지급액 즉시 안내
- **💰 수령액 계산기** — 거주 지역·계층·가구원 수 입력 → 1인/가족 합산 수령액 자동 계산
- **🏥 건강보험료 기준 대상 판정** — 2026년 3월 부과 건강보험료 본인부담금으로 소득 하위 70% 여부 즉시 확인 (직장/지역/혼합 가입유형 · 외벌이/맞벌이 분기 처리)
- **💡 월급으로 건보료 추정** — 본인 월급 입력 시 직장가입자 건보료율(7.09%)로 본인부담분 자동 환산
- **📅 신청 요일(5부제) 안내** — 출생연도 끝자리 입력 시 1차/2차 신청 요일 즉시 표시 (5/1 근로자의 날 처리 포함)
- **⚠️ 스미싱 경고** — URL 포함 문자·알림이 100% 사기임을 상단/하단에서 반복 안내
- **📤 결과 공유** — 클립보드 복사로 가족·지인에게 결과 공유

## 🛠 기술 스택

- **순수 정적 HTML** — 단일 `index.html` 파일 (CSS·JS 인라인)
- **Kakao Maps JavaScript SDK** — 시군구 폴리곤 렌더링, 좌표→행정구역 매핑
- **southkorea-maps GeoJSON** — KOSTAT 2018 시군구 경계 데이터
- **Vercel** — 정적 호스팅
- **쿠팡 파트너스** — 광고 슬롯 (캐러셀 형식)

## 🚀 로컬 실행

별도 빌드 도구 없이 브라우저에서 바로 실행 가능합니다.

```bash
# 저장소 클론
git clone https://github.com/gongdly/Oil-Relief-Fund.git
cd Oil-Relief-Fund

# 정적 서버로 실행 (Python 3 기준)
python3 -m http.server 8000

# 또는 Node 사용 시
npx serve .
```

브라우저에서 `http://localhost:8000` 접속.

> **⚠️ 카카오맵 API 키 도메인 설정**: 로컬에서 지도가 표시되지 않으면 Kakao Developers 콘솔에서 `localhost` 도메인을 허용 도메인에 등록해야 합니다.

## 📊 지급 기준 (정부 공식 발표 기준)

### 1인당 지급액 (계층 × 지역)

| 계층 \ 지역 | 수도권 | 일반 비수도권 | 우대지역 | 특별지역 |
|---|---:|---:|---:|---:|
| 일반 (하위 70%) | 10만원 | 15만원 | 20만원 | 25만원 |
| 차상위·한부모 | 45만원 | 50만원 | 50만원 | 50만원 |
| 기초생활수급자 | 55만원 | 60만원 | 60만원 | 60만원 |

- **인구감소 우대지역**: 49곳
- **인구감소 특별지역**: 40곳

### 2026년 3월 건강보험료 기준 (외벌이, 본인부담금)

| 가구원 수 | 직장가입자 | 지역가입자 | 혼합 |
|---:|---:|---:|---:|
| 1인 | 130,000원 | 80,000원 | — |
| 2인 | 140,000원 | 120,000원 | 150,000원 |
| 3인 | 260,000원 | 180,000원 | 240,000원 |
| 4인 | 320,000원 | 220,000원 | 300,000원 |
| 5인 | 390,000원 | 270,000원 | 360,000원 |
| 6인 이상 | 430,000원 | 300,000원 | 400,000원 |

- 맞벌이·다소득원 가구는 외벌이 기준에서 **가구원 수 +1명** 적용
- 장기요양보험료는 제외

### 고액자산가 컷오프 (가구 합산 기준)

다음 중 하나에 해당하면 가구원 전원 지급 제외:
- 2025년 재산세 과세표준 12억원 초과 (1주택 기준 공시가 약 26.7억원)
- 2024년 귀속 금융소득 2,000만원 초과

### 신청 일정

| 차수 | 기간 | 대상 |
|---|---|---|
| 1차 | 2026.4.27 ~ 5.8 | 기초생활수급자·차상위·한부모 |
| 2차 | 2026.5.18 ~ 7.3 | 소득 하위 70% 일반 가구 + 1차 미신청 취약계층 |

- **사용 기한**: 2026.8.31 (미사용 잔액 전액 소멸)
- **이의신청**: 2026.5.18 ~ 7.17
- **첫 주 5부제**: 출생연도 끝자리 기준 요일제 적용
- **전담콜센터**: ☎1670-2626 / 일반 ☎110
